# sourcecode-analysis

每2周对一个热门的前端框架或库进行源码分析，并画出思维导图

起因：前端业务做得越来越索然无味，只有源码才能激起自己对知识的渴望，对架构的追求，一点一滴，滴水穿石~~

类似项目：[stonehank 的 sourcecode-analysis](https://github.com/stonehank/sourcecode-analysis)
